# Portföljtrack / AktieX

Ett webbaserat projekt byggt med React, TypeScript och TailwindCSS.  
Syftet är att simulera en aktiemarknad där användaren kan köpa och sälja aktier, följa sin portföljutveckling och justera inställningar.

# Funktioner
- Simulering av aktiemarknaden
- Stressnivå och chans för börskrascher
- Hantering av portfölj (köp, sälj, återställ)
- Inställningar (saldo vid start, marknadens hastighet)
- Responsiv design med TailwindCSS
- Dark mode

# Teknologier
- React
- TypeScript
- TailwindCSS
- Vite

# Installation
### Klona projektet

git clone https://github.com/AlbinWr/portfoljtrack.git
cd portfoljtrack

### Gå till projektmapp

cd portfoljtrack

### Installera packages

npm install

### Starta utvecklingsservern

npm run dev
